# AI Content Factory — Blog Post Template

> 用途：使用 AI 快速生成高质量 SEO 友好博客文章  
> 适用 Niche：AI & Productivity Tools（推荐）、Personal Finance、Creator Economy  
> 预计字数：1500–3000 字 | 阅读时间：5–10 分钟

---

## 📋 模板结构

```
1. 标题 Hook（H1）
2. 简介/导言（150字以内）
3. 目录（Table of Contents）
4. 主体章节（H2 × 4–6）
   - 每章包含：概念解释 + 案例/数据 + 实操建议
5. FAQ 章节（可选，SEO 加分）
6. 结论 + CTA
```

## 🎯 AI 填写指令（Prompt）

### Step 1：生成文章框架

```markdown
请为以下主题生成一篇博客文章框架：

主题：[YOUR TOPIC]
目标受众：[AUDIENCE DESCRIPTION]
核心信息：[KEY MESSAGE]
SEO 关键词：[PRIMARY KEYWORD] + [2-3 SECONDARY KEYWORDS]
文章长度：[SHORT (800字) / MEDIUM (1500字) / LONG (3000字)]

要求：
- 使用 [TARGET AUDIENCE] 的语言风格
- 包含 1 个争议性观点引发讨论
- 每个章节需要有"实操建议"或"可操作步骤"
- 结尾包含明确的 CTA（Call to Action）
```

### Step 2：生成完整文章

```markdown
请基于以下框架，撰写完整的博客文章：

[PASTE FRAMEWORK HERE]

写作要求：
- 语气：[CONVERSATIONAL / PROFESSIONAL / CASUAL]
- 开头：使用故事、问题或惊人数据引入（避免"在这篇文章中..."式开头）
- 章节标题：使用 H2，包含数字或问题（如"5 Ways to..."、"Why You Should..."）
- 每个 H2 至少包含 2 个段落
- 添加 1-2 个小标题（H3）
- 段落之间使用过渡句
- CTA 类型：[SUBSCRIBE / TRY_TOOL / SHARE / DOWNLOAD]
- 避免被动语态
- 使用简短句子，每段不超过 4 句
```

### Step 3：生成配套内容

```markdown
请为上述文章生成：

1. **Meta Description**（150字符以内，含主关键词）
2. **SEO 优化建议**：
   - 内部链接建议（目标页面）
   - 外部链接建议（权威来源，3个）
   - 图片 alt 文本建议（3个）
3. **CTA 变体**（3个不同风格的行动号召）
4. **Email 推广文案**（50字以内，用于 newsletter）
5. **1 个 Quora/Reddit 选题**（适合在该平台引发讨论）
6. **相关长尾关键词**（10个）
```

## ✅ 质量检查清单（发布前必读）

- [ ] 标题包含主关键词，长度 < 60 字符
- [ ] Meta description 包含关键词，< 160 字符
- [ ] 文章至少 1,500 字（H2 × 4 以上）
- [ ] 每个章节有至少 1 个具体例子或数据
- [ ] 无被动语态过多
- [ ] CTA 清晰且只有一个主行动
- [ ] 内部链接：至少 2 个
- [ ] 外部链接：至少 1 个权威来源（.gov / .edu / 知名媒体）
- [ ] 图片：每 300 字至少 1 张（带 alt 文本）
- [ ] URL slug：简短、含关键词、连字符分隔
