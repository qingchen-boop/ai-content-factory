# AI Content Factory — Landing Page Template

> 用途：高转化率落地页，适配 Lead Magnet / 产品发布 / 课程销售  
> 核心框架：AIDA + Social Proof + urgency  
> 适用渠道：SEO 落地页、付费广告、Newsletter CTA、社交媒体链接

---

## 📋 页面结构

```
[1] Hero Section（英雄区）
    - Headline（主标题）
    - Sub-headline（副标题）
    - Primary CTA（主 CTA）
    - Trust Badge / Social Proof Bar

[2] Problem Section（痛点）
    - 痛苦描述
    - "Sound Familiar?" 问题清单

[3] Solution Section（解决方案）
    - 产品/服务介绍
    - 核心价值主张
    - 3–5 核心卖点

[4] How It Works（操作流程）
    - 3 步上手（透明化）

[5] Social Proof（社会证明）
    - Testimonials（用户评价）
    - Case Study（可选）
    - Numbers/Stats（数据背书）

[6] Pricing / Offer Section（定价/优惠）
    - 价格锚定
    - What's Included（包含内容）
    - Guarantee（保证）

[7] FAQ（常见问题）
    - 5–7 个核心问题

[8] Final CTA（最终行动召唤）
    - Urgency element
    - 最后一个转化机会
```

---

## 🎯 AI 填充指令（Prompt）

### 生成完整 Landing Page

```markdown
请为以下产品/服务生成完整 Landing Page 文案：

产品名称：[PRODUCT NAME]
核心价值：[CORE VALUE PROPOSITION]
目标受众：[AUDIENCE DESCRIPTION]
解决的问题：[PROBLEM IT SOLVES]
理想结果：[DESIRED OUTCOME]
定价：[PRICE] / [ALTERNATIVE PRICING]
竞品对比（如果有）：[COMPETITORS]

文案要求：
- Headline：≤ 10 词，含数字或强情绪词，能独立存在
- Sub-headline：≤ 25 词，补充"如何"和"为什么"
- Body：每个 Section ≤ 100 词
- CTA：动词开头，明确利益，避免"Submit"
- 语气：[CONFIDENT / URGENT / WARM / DIRECT]

每个 Section 需要：
1. Hero：Headline + Sub + CTA + Social proof bar
2. Problem：3 个 pain points + "Sound familiar?"
3. Solution：1 句话介绍 + 5 核心卖点（icon style）
4. How It Works：3 步流程
5. Social Proof：3 个 testimonials + stats
6. Pricing：3-tier 或 single tier 均可
7. FAQ：5 个真实 objection + 诚实回答
8. Final CTA： urgency + CTA
```

### 生成 A/B 测试 Headline

```markdown
请为 [PRODUCT] 生成 5 个 A/B 测试 Headline：

产品：[PRODUCT NAME]
主要利益：[KEY BENEFIT]
目标受众：[AUDIENCE]

要求：
- 每个 Headline ≤ 10 词
- 风格各异（数字型/问题型/命令型/故事型/反常识型）
- 含目标关键词
- 附加 1 行 Sub-headline 建议
```

## 📐 页面设计建议（文字布局参考）

| 元素 | 最佳实践 |
|------|----------|
| Headline | 左对齐或居中，≥ 48px 字号，主色或深色背景 |
| CTA Button | 高对比色（主色 on 白/浅灰），≥ 44px 点击区域，悬停动效 |
| Trust Bar | 灰度图标，低调，放在主 CTA 正下方 |
| Section Spacing | 每个 Section ≥ 60px padding，避免拥挤 |
| Mobile | 单列布局，CTA 固定在底部（sticky）|
| Images | 产品截图/头像/abstract 配图，无版权风险 |

## ✅ 发布前检查清单
